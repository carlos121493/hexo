2017-02-04

发表于 2017-02-04   |  

昨天和发小见了一下没什么其它事情，今天一天又全无学习的感觉。
看了点书。很感慨。总是太忙的时候感觉太累，太自由了自己又无法把控, 这真是需要继续研究的课题。

听说二姑已经将一起看的书看完了。而我才看了三章，自惭形秽。
感觉自己有状态域值，好的时候就多做点，不好的时候就娱乐些。然而过多的娱乐会让自己状态变得差起来。

看了风雨哈佛路才体会到，主人公的生活虽然不堪，却有着极其敏锐的观察力和对生活的体会感，在一种爱的生活中对生活的艰辛他人的表达有所感受。
整体上就像看小说，看了很多，停下来有很多收获。和大部分自传不一样。

今天继续看7-11的自传，这种书看得时候每看一点就要思考很多，联系过去和未来不断思考借鉴。这是另外一种体验。
然而方法和体会是不同效果的学习，体材不同的书也得用不同的方法不同的时期来看。

在7-11里主人公的事业随着消费者对品牌的认知，对其特点和要求也进一步增加。
所以整个在发展过程中这种创新需要不断的进行。而围绕着这一点的价值输出却是一致的。非常佩服这样的方式和做法。
对于消费者的价值认知和表现也有相当深入的分析。并不随着外在的变化而变化，不随着大家的观点及分析，而是靠自己假设并验证了这一有效性。

比如日本在物价上涨，消费低迷，税收增加，多老少子时。消费者的心理和今天的中国有很大的相似型。
虽说社会上都会觉得物价在上涨，会觉得有优惠券的需求。但是很多降价却是伪需求。
而更多的则是用数据验证了的新鲜感，以及对品质的要求。

在杭州时经常吃的便宜的东西，看看不禁感慨好便宜啊，甚至有点不相信，一般却会到旁边装修比较好评价比较不错的馆子里，点一些价格比较适中的套餐。
毕竟出门在外得吃顿好的，一旦吃到好的会连续吃几次，直到突然发挥不稳定或者有点腻了才会转变。然而一些看似便宜的外卖，每次点到虽然有种点了虽然很实惠，但是一般都因为感觉实在不满意下次不会点了。
请客吃饭看面子，累了或者心情不快都会想吃点好的。基本上会过滤掉价格特别低的和价格特别高的菜品部分。
所以在今天的社会还是，新鲜和品质才是非常重要的部分。价格当遇到对价值认知的时候，也会妥协的，毕竟这样对大家都好。

今天最让我感到神奇的是。晚上一家人在奶奶家里吃饭。探讨如何邀请爸爸去参加婚礼的情况。大家群策群力一起想办法说服了老爸。
虽然我没体会到我说话能有这样的效果，作为旁观者的大家对待我们的关系认识的比较深入。而且对我的表达方式也做了非常好的优化，居然一步到位了。
有一家人的支持和帮助，让我深感幸福。年轻时的自己心理压力和看问题的角度都非常的有限，造成对待事物的方式非常谨小慎微且诚惶诚恐。
有幸现在一家人的帮忙能让我过得更好，感谢这一切。
